# Data

This directory contains all datasets and data documentation associated with the project.

## Recommended Structure

data/

├── raw/

├── processed/

└── external/

## Definitions

raw/
Original datasets obtained directly from the source.

processed/
Cleaned and transformed datasets used for analysis and model training.

external/
Auxiliary datasets, metadata, or third-party resources.

## Documentation Requirements

For each dataset provide:

- Dataset name
- Source
- License
- Number of samples
- Number of features
- Preprocessing steps

## Reproducibility

All preprocessing operations should be documented and reproducible.
