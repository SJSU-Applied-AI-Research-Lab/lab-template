# Results

This directory stores all experimental outputs generated during the project.

## Contents

- Performance metrics
- Evaluation summaries
- Figures
- Tables
- Confusion matrices
- ROC curves
- Error analyses

## Recommended Structure

results/

├── figures/

├── tables/

└── metrics/

## Reproducibility

Figures and tables appearing in manuscripts should be traceable to files stored within this directory.

## Publication Policy

Results reported in publications should be generated directly from reproducible evaluation scripts.
