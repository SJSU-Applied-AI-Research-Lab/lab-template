# Source Code

This directory contains the reusable source code used throughout the project.

## Purpose

The src directory should contain:

- Data preprocessing pipelines
- Feature engineering modules
- Model training scripts
- Evaluation functions
- Inference pipelines
- Utility functions

## Recommended Structure

src/

├── preprocessing.py

├── train.py

├── evaluate.py

├── inference.py

└── utils.py

## Best Practices

- Use meaningful function names.
- Document all major functions.
- Keep code modular and reusable.
- Avoid placing research code exclusively in notebooks.
- Ensure scripts can be executed independently.

## Reproducibility

All experiments reported in publications should be reproducible from the source code contained in this directory.
